# NeuroFace Analyzer

A web-based application that analyzes facial features to detect potential health conditions using computer vision and machine learning.

## Features
- Real-time facial analysis using webcam
- Blink rate monitoring
- Facial asymmetry detection
- Micro-expression analysis
- Health condition prediction
- PDF report generation
- User authentication and profile management

## Tech Stack
- **Backend**: Flask, SQLAlchemy
- **Frontend**: HTML, CSS, JavaScript
- **Computer Vision**: OpenCV, MediaPipe
- **Database**: SQLite
- **Authentication**: Flask-Login
